A simple plugin to add links to your social network. You can add the list on header, footer or widget.

== Description ==

This plugin simply displays links to your social netwrok.
You can add the links in header section, footer section or in the sidebar using widgets.

== Installation ==
1. Download the Plugin using the Install Plugins 
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Setting > Social Links
4. Add the complete Urls. e.g. https://www.facebook.com/sanskrutitech
5. Select Header / Footer to display the Plugin
6. You may also add the links using Widget "Social Link"
